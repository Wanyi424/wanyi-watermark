使用说明

三步搞定，无需登录
1. 粘贴链接
把抖音/小红书的分享链接（或整段分享文案）粘贴到输入框，支持通用平台兜底。
2. 获取信息
拿到无水印视频（可在线播放/下载），或图文图集（小红书提供 PNG 高清 / WebP 轻量）。无需 API Key。
3. 提取文案
识别视频语音生成文字稿，支持复制与导出 Markdown。需配置 API Key。

手动复制

当前环境无法自动复制

检测到当前处于非安全上下文（如通过局域网 IP 访问），浏览器已禁用一键复制。请在下方文本框中按 Ctrl/⌘ + C 手动复制；如需恢复一键复制，请改用 localhost 或 HTTPS 访问。
